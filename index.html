使用说明

1. 初始录入：
点击“初始录入”，可批量粘贴数据（支持 A1 50 或 JSON 格式），快速建立库存。

2. 消耗与补货：
• 消耗：选中色号 -> 点击底部“录入粒数” -> 输入粒数自动扣除，按1g=100粒计算。
• 可直接点击数字修改库存重量，系统会自动根据重量变化记录为“补货或消耗”。
• 也可点击+10g按钮快速补货。

3. 阈值提醒：
在顶部设置阈值（如5g），低于该值的色号会显示红色背景。点击“补货单”可查看所有缺货色号。

4. 忽略清单：
• 设置方法：选中色号后，点击底部的 “🔔 开启阈值” 或 “🔕 关闭阈值” 即可批量设置。
• 关闭阈值的色号，右侧会显示灰色 🔕 图标，不显示红色低库存警告，也不计入“补货”统计。
• “补货”统计和“补货单”会自动过滤这些色号。

5. 历史记录：
点击“查看明细”按钮可查看色号的库存明细，支持删除记录，删除后库存会自动回滚。

6. 数据备份：
点击“数据备份”可导出包含所有记录和设置的完整备份。更换设备时，在恢复页面粘贴该备份即可无缝迁移。
